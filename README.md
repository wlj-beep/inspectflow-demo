# InspectFlow

Manufacturing Inspection System — Management Demo

**Live demo:** https://wlj-beep.github.io/inspectflow-demo/

## About

InspectFlow is a browser-based digital inspection data collection system, built to replace paper-based manufacturing inspection records. This repository contains a fully functional management demo.

Built with React 18 + Vite. No backend — all state is in-memory.

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
